# BuildOS — Карта экранов и User Flows (MVP)

## 1) Основные экраны (10–15)
1. Login / Register
2. Выбор компании (если несколько)
3. Dashboard (обзор объектов, прибыль, статусы)
4. Объекты — список
5. Объект — обзор (таймлайн, этапы, финансы)
6. Помещения объекта
7. Этапы работ (workflow)
8. Калькулятор сметы
9. Версии сметы (сравнение)
10. Материалы объекта
11. Фото и прогресс
12. Команда и назначения
13. Документы и счета
14. Клиентский кабинет
15. Настройки компании

## 2) Ключевые user flows
### Создание объекта
Dashboard → Объекты → Создать → Тип объекта → Помещения → Шаблон этапов → Готово

### Смета
Объект → Смета → Ввод параметров → Расчёт → Версия v1 → PDF → Клиент

### Работа прораба
Объект → Этап → Назначить работников → Загрузить фото → Статус

### Клиент
Личный кабинет → Просмотр КП → Статусы → Фото → Оплата
