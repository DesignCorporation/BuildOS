# BuildOS — Шаблоны объектов (v0)

## Назначение
Шаблоны позволяют создать объект и получить первичную смету за 2–5 минут без ручного ввода всех работ.

Каждый шаблон:
- создаёт помещения
- подключает этапы работ
- подставляет типовые работы и нормы
- служит базой для кастомизации

---

## 1) Квартира (ремонт)

### Типовые параметры
- Площадь: 30 / 40 / 50 / 60 / 80 м²
- Высота потолка: 2.5 / 2.7 / 3.0 м
- Состояние: новостройка / вторичка

### Типовые помещения
- Коридор
- Кухня
- Санузел
- Гостиная
- Спальня (×1–2)
- Балкон (опц.)

### Автоматически добавляемые этапы
- Демонтаж (если вторичка)
- Черновые работы
- Инженерия
- Чистовая отделка
- Сдача

### Примеры авто-работ
- Стяжка пола (м²)
- Штукатурка стен (м²)
- Электроточки (шт)
- Плитка санузел (м²)
- Покраска/обои (м²)

---

## 2) Дом (строительство / ремонт)

### Типовые параметры
- Площадь дома: 80 / 100 / 120 / 150 / 200 м²
- Количество этажей
- Участок: да/нет

### Типовые этапы
- Геодезия
- Котлован
- Фундамент
- Коробка
- Крыша
- Инженерия
- Отделка

### Помещения
- Технические зоны
- Жилые комнаты
- Санузлы
- Лестницы

---

## 3) Коммерческий объект

### Типы
- Офис
- Магазин
- Кафе/ресторан

### Особенности
- Повышенные коэффициенты
- Доп. требования (пожарка, санитария)

---

## 4) Логика работы шаблонов
1. Пользователь выбирает шаблон
2. Вводит ключевые параметры
3. Система создаёт:
   - объект
   - помещения
   - этапы
   - базовую смету v1
4. Пользователь редактирует

---

## 5) Польза для бизнеса
- Быстрое КП
- Меньше ошибок
- Стандартизация
- Лёгкое обучение новых менеджеров
